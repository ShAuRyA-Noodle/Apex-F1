## 1. Pre-flight clearance (do FIRST — 10 min)

Skipping clearance risks paying the $350 and being rejected for a confusingly-similar mark. Three searches:

### 1.1 USPTO TESS search
- URL: https://tmsearch.uspto.gov/bin/gate.exe?f=login&p_lang=english&p_d=trmk
- Click **Basic Word Mark Search (New User)**
- Search Term: `APEX`
- Field: `Word Mark` (or use the Combined Word Mark Search)
- Live Marks Only: Yes
- Look for: any existing **live registered** mark in Class 41 (entertainment), Class 38 (telecom), Class 9 (downloadable software), Class 25 (apparel)
- Expected: many "Apex" marks exist in unrelated classes. What you need to confirm is **no live registered Apex mark in Class 41 covering motorsport / sports media / online entertainment**. There are several "Apex" marks (e.g. Apex Legends video game) — game ≠ motorsport-news platform, but the proximity is real. Read the Goods & Services description on any Apex mark you find and screenshot it for your filing notes.

### 1.2 EUIPO eSearch plus
- URL: https://euipo.europa.eu/eSearch/
- Search Term: `APEX`
- Class: 41
- Status: filed + registered

### 1.3 UKIPO trademark search
- URL: https://www.gov.uk/search-for-trademark
- Search Term: `APEX`
- Classes: 41

**If no blocking mark in Class 41 motorsport/sports-media:** proceed. **If blocking:** consider Apex + descriptive suffix ("Apex Paddock", "Apex Pitwall") and re-clear.

## 2. Filing (TEAS Plus — 20 min)

USPTO has 3 filing tiers. TEAS Plus is the cheapest at **$350/class** but requires using the USPTO's pre-approved Goods & Services descriptions verbatim.

### 2.1 Create USPTO MyUSPTO account
- URL: https://account.uspto.gov/
- Use a permanent founder email you control forever (not Gmail throwaway — file under a custom-domain email like `tm@apex.gg` if domain is registered, otherwise personal)

### 2.2 Start TEAS Plus filing
- URL: https://www.uspto.gov/trademarks/apply/teas-plus
- Click **Begin New Application**

### 2.3 Form fields (paste verbatim where possible)

| Field | Value |
|---|---|
| Filing Basis | Section 1(b) — Intent to Use |
| Mark Type | Standard Character Mark |
| Mark Literal Element | `APEX` |
| Applicant Type | Individual (or your future LLC if formed; can be assigned later) |
| Applicant Name | Your legal name |
| Applicant Address | Your registered home/business address |
| Applicant Citizenship | India (or relevant) |
| Applicant Email | `tm@apex.gg` (preferred) or your personal |
| Class | 041 |
| Identification of Goods/Services | **Use these USPTO-approved IDs (TEAS-Plus pre-approved list — copy/paste verbatim):** `Entertainment services in the nature of providing on-line non-downloadable videos in the field of motor sports; providing a website featuring sports news in the field of automobile racing; providing online information in the field of automobile racing` |

### 2.4 Optional additional classes (skip for now, file later from revenue)

You **do not** need to file Class 9 (downloadable apps), Class 38 (telecom), or Class 25 (apparel) on Day 1. Each additional class = +$350. File when:
- Class 9: when mobile app submits to App Store / Play Store (≈ M9 per master plan)
- Class 25: when merch store launches (≈ M12+)
- Class 38: not needed for this product

### 2.5 Signature + payment

- Receipt: **save the PDF receipt + Serial Number to a permanent folder**
- USPTO acknowledgment email arrives within 1 hour with the Serial Number
